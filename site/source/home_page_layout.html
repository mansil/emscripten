Emscripten é uma "cadeia de ferramentas" para compilar para asm.js e WebAssembly, criado utilizando LLVM, que o deixa executar C e C++ na Web na velocidade quase nativa sem plug-is

Portabilidade

Compile os seus projetos existentes escritos em C ou C++ e execute-os em todos os navegadores modernos..

APIs

Emscripten converte OpenGL em WebGL e permite utilizar APIs familiares, tais como SDL ou HTML5 diretamente.

Rápido

Graças ao LLVM, Emscripten, asm.js e WebAssembly, o código é executado em velocidade quase nativa.
